# Webpack

Webpack 是一个开源的前端打包工具。

Webpack 提供了前端开发缺乏的模块化开发方式，将各种静态资源视为模块，并从它生成优化过的代码。

使用 Webpack 可以从终端、或是更改 webpack.config.js 来设置各项功能。
