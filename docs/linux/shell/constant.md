# Shell 常量

| 常量 | 描述                                                                                  |
| :--- | :------------------------------------------------------------------------------------ |
| \$0  | 当前脚本的文件名                                                                      |
| \$n  | 传递给脚本或函数的第 n 个参数值，n=1..9                                               |
| \$\* | 传递给脚本或函数的所有参数，此选项参数可超过 9 个                                     |
| \$#  | 传递给脚本或函数的参数个数                                                            |
| \$\$ | 当前 Shell 进程 ID。对于 Shell 脚本，就是这些脚本所在的进程 ID                        |
| \$@  | 跟\$\*类似，但是可以当作数组用                                                        |
| \$?  | 执行上一个指令的返回值 (显示最后命令的退出状态。0 表示没有错误，其他任何值表明有错误) |
| \$!  | 后台运行的最后一个进程的进程 ID 号                                                    |
| \$-  | 显示 Shell 使用的当前选项，与 set 命令功能相同                                        |
