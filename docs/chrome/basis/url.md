# Chrome URLs 列表

借助 Chrome URLs 列表不仅可以帮助您访问 Chrome 的隐藏功能，还可以让您在遇到任何问题时进行调试。

| 地址             | 描述            |
| :--------------- | :-------------- |
| <chrome://about> | Chrome URL 列表 |

## <chrome://version/>

它提供了有关 Chrome 版本的详细信息，包括内部版本号、用户代理、Flash 支持、渲染引擎等。您还可以找到苹果的 WebKit 和 Mozilla 的 Gecko 内部版本号。
