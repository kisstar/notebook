# 计算机

计算机（computer）俗称电脑，是现代一种用于高速计算的电子计算机器，是能够按照程序运行，自动、高速处理海量数据的现代化智能电子设备。

通过计算机可以进行数值计算和逻辑计算，同时还具有存储记忆功能。
